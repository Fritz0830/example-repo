Financial Calculator



Description



A Python command-line tool that serves as a financial calculator with two products:

Bond and Investment. The user selects a product type and inputs the relevant

financial details, and the script calculates and returns the appropriate financial output.





What It Does



Investment Calculator

Calculates the future value of an investment based on:

* the initial amount invested
* the interest rate (%)
* the number of years
* simple or compound interest



Simple Interest Formula: initial amount invested × (1 + interest rate/100 × years)

Compound Interest Formula = Initial amount invested × (1 + interest rate/100) ^ years





Bond Calculator

Calculates the monthly repayment amount on a home loan based on:

* the home loan amount
* the interest rate
* repayment period in months



Monthly Repayment Formula = (interest rate/12/100 × loan amount) / (1 - (1 + interest rate/12/100) ^ -months)
